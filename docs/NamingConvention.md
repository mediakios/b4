# Naming Convention

## Modules

PascalCase:

```text
KnowledgeModel
KnowledgeLoader
KnowledgeEngine
```

## Public Subs

PascalCase:

```text
Initialize
LoadKnowledge
Analyze
```

## Local Variables

Descriptive names:

```text
NormalizedText
CandidateList
FeatureMap
```

## Types

Prefix with `T`:

```text
TEvidence
TRule
TInferenceRule
```
