# Modules

## 1. KnowledgeModel.bas

Central Types and runtime collections.

## 2. KnowledgeLoader.bas

Loads knowledge files and validates structure.

## 3. KnowledgeNormalizer.bas

Normalizes user input and token variants.

## 4. KnowledgeSemantic.bas

Extracts semantic categories and features.

## 5. KnowledgeRule.bas

Evaluates keyword and rule matches.

## 6. KnowledgeInference.bas

Evaluates feature-based inference rules.

## 7. KnowledgeEvaluator.bas

Combines all evidence and selects candidates.

## 8. KnowledgeResult.bas

Builds deterministic output objects and text.

## 9. KnowledgeRuntime.bas

Coordinates module initialization and execution.

## 10. KnowledgeEngine.bas

Public API for host applications.
